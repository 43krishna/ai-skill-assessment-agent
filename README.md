# AI Skill Assessment & Learning Agent

## Overview
This project is an AI-powered agent that evaluates a candidate’s skills based on a job description and generates a personalized learning plan.

## Features
- Extracts key skills from job description
- Generates assessment questions
- Evaluates answers using rule-based scoring
- Identifies skill gaps
- Generates a personalized learning roadmap

## Tech Stack
- Python
- Streamlit

## How to Run
```bash
pip install streamlit
streamlit run gp.py
